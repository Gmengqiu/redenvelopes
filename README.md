# 简介
马上过年了，又到了发红包的时间，抢红包又蠢蠢欲动了，所以利用Android无障碍服务实现了一个微信自用抢红包辅助，好快速抢家里长辈发的红包哈哈。


# 使用
打开红包辅助，打开无障碍，通知权限，然后就可以了
